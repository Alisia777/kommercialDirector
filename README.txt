Кейсы (картинки)

Положите ваши скрины/фото в эту папку.
Рекомендации:
- Формат: .png или .jpg
- Ширина: 1200–1600px (так будет чётко и быстро)
- Названия: case-01.png, case-02.png, ...

Далее откройте cases.html и замените ссылки вида:
assets/cases/placeholder.svg
на ваши реальные файлы.

Если хотите, можно хранить все кейсы прямо в GitHub репозитории (public), а сайт на GitHub Pages.

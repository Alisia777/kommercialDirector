# Обновление сайта (v5)

Этот архив содержит **обновлённые файлы сайта** (новые вкладки **Кейсы / Услуги / Демо / Telegram** + плавные анимации при скролле + калькулятор `xgrow_planner_calc.html` + Яндекс.Метрика).

## Как поставить в репозиторий GitHub Pages
1) Открой репозиторий `kommercialDirector` (ветка `main`).
2) **Заменой** скопируй в корень репозитория файлы из этого архива:
   - `index.html`
   - `services.html`
   - `cases.html`
   - `demos.html`
     - `tg.html`
   - `xgrow_planner_calc.html`
   - папку `assets/`
3) **Не удаляй** свои изображения/доки (png/pdf) — они уже лежат в репо и используются в кейсах.
4) Сделай commit → дождись деплоя GitHub Pages (обычно 1–2 минуты).
